STRUCTURE DU REPOSITORY FAUSTPLAYGROUND

**** Les 2 versions du playground *****
index.html
pedagogie/
	index.html

**** Les Scripts ****
js/
	analyser.js					--> historique Web Audio Playground
	AudioContextMonkeyPatch.js 	--> historique Web Audio Playground
	Connect.js					--> historique modifié Web Audio Playground
	Dragging.js					--> historique modifé Web Audio Playground
	EquivalentFaust.js			--> fonctionnalité du calcul de faust equivalent
	Export.js					--> Fait appel à ExportLib.js & qrcode.js qui sont plus haut dans la hiérarchie
	Main.js						--> Point d'entrée de l'application & fonctionnalités générales de DnD
	Utilitary.js				--> Functions "utilitaires"
	
	Nodes/
		NodeClass.js			--> Toutes les fonctions liées aux modules
		FaustModuleInterface.js	--> Parse l'interface JSON et crée l'interface associée
	
	Pedagogie/
		Library.js				--> Bibliothèque d'objets pour la version pédagogique
		PedagogieTools.js		--> Tooltips pour la version pédagogique
	
	Save/
		Saver.js				--> Save/Recall scene sous forme de JSON
	
	Scenes/
		SceneClass.js			--> Toutes les fonctions liées à une scène
		Accueil.js				--> Menu d'accueil de la version pédagogique
		Pedagogie.js			--> Scène de la version pédagogique
		Finish.js				--> Menu d'export de la version pédagogique
		Playground.js			--> Scène de la version normale
		
**** Les Styles ****
css/
	Accueil.css					--> style lié au menu Accueil
	Finish.css					--> style lié au menu Finish
	FaustModules.css			--> styles des interface Faust
	Library.css					--> style lié à la bibliothèque latérale
	Main.css					--> style de base
	Nodes.css					--> style des noeuds, I/O et connexions
	Pedagogie.css				--> style lié à la scène pédagogique
	Playground.css				--> style lié à la scène "normale"
	Reset.css					--> historique Web Audio Playground
	Scenes.css					--> style de la scène, ses titres, etc
		
**** Autres ****
archive/						--> historique Web Audio Playground
fonts/							--> historique Web Audio Playground
img/							--> toutes les img utilisées
